# Data Attributions

Poke Pal uses data from these community sources:

- **PvPoke** (pvpoke.com) — Pokemon rankings, movesets, and meta analysis. PvPoke is open-source (GPL). Rankings data is fetched and processed for the league ranker feature.
- **LeekDuck** (leekduck.com) — Current raid bosses and Team GO Rocket lineups. Data refreshed locally and committed.
- **Pokemon GO** is a trademark of Niantic, Inc. Poke Pal is a fan-made companion tool and is not affiliated with or endorsed by Niantic.

If you maintain one of these sources and have concerns, please reach out via GitHub issues.
