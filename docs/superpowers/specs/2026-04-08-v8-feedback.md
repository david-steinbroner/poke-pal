# v0.8.0 Feedback — Founder Notes (2026-04-08)

All items from David's testing of v0.7.0. Each must be addressed.

## Navigation

1. **League tab memory:** When on Leagues, select a league, then tap another tab (Teams), then tap Leagues again — should return to the selected league page, NOT the leagues list.

2. **Teams tab persistence:** When on Teams and a Pokemon is added to e.g. Great League team, then user taps a different league pill (Ultra), then taps back to Great — the Pokemon should still be there.

3. **Remove back button from Teams page.** Teams is reached via tabs and other entry points. No back button needed.

## CopyBar Redesign

4. **Kill the search string display.** Don't show the actual string — it takes up space and isn't useful. PokeGenie doesn't show it either.

5. **New CopyBar component — "CopyButton":** A full-width button (or half-width when paired). Text: "Copy Search String". On tap: "Copied — paste in game!" for 3 seconds.

6. **Dual CopyButton layout:** New component that holds up to 2 CopyButtons side by side on one line, evenly spaced.

7. **League page dual buttons:**
   - Left button: always "Copy Meta Pokemon" (the full meta search string)
   - Right button: appears when 1+ Pokemon added to team — "Copy Your Team"
   
8. **Counter page dual buttons:**
   - Left: "Copy Counters" (counters search string)
   - Right: "Copy Shadow Counters" (shadow search string)

9. **Remove Clear button** from the Your Team section on the home page.

11. **Pokemon pills must fit on ONE line** — no wrapping to second line. Truncate or scroll if needed.
